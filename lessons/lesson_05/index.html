Lesson 05 (заголовок fixed)

Свойство position определяет, какой будет использован метод для расположения элемента (static, relative, fixed, absolute, sticky)

  • static - свойство position go по умолчанию для всех элементов, когда элементы ведут себя привычно для нас без изменений
  • fixed - вырывает элемент из потока и делает его невидимым для других элементов
  • sticky - вначале элемент сохраняет место, но при указании в пикселлях значений по осям (top, left, right, bottom) уходит из потока элементов
  • absolute - по умолчанию двигает элементы относительно body через свойства (top, left, right, bottom)
  • relative - вырывается из потока, но оставляет за собой место и свидгается относительно него. Обычно используется в связке с absolute, чтобы указывать позиционирование для родителя.
