Lesson_03. CSS box-model

CSS box-model

Составные элементы контейнера:

  • Margin - внушний отступ
  • Border - граница элемента
  • Padding - внутрений отступ
  • Content - содержимое блока

Значения для margin и padding:

  • Top - margin-top
  • Right - margin-right
  • Left - margin-left
  • Bottom - margin-bottom
