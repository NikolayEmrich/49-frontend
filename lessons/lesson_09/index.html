Lesson_09. DOM intro

DOM - Document Object Model

Все элементы на странице можно представить в виде глобального объекта document.

Чтобы находить элементы в DOM нужно доабвить им что-то уникальное (класс или id):

  • Поиск по тэгам - document.getElementsByTagName()
  • Поиск по классам - document.getElementsByClassName()
  • Поиск по id - document.getElementById()
  • Универсальный метод поиска. Класс уточняется через точку. ID через решетку. В ответе придет первый элемент удовлетворяющий поиск - document.querySelector()
