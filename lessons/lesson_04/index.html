Lesson_04. CSS flexbox

Flexbox - гибкая технология для позиционирования элементов в CSS

Правила работы с flexbox:

  • display:flex - включиние flex-отображения в контейнере нужно задать свойство display:flex
  • Justify-content - свойство выравнивания по главной оси
  • align-items - свойство выравнивания по поперечной оси
  • gap - значение промежутка между flex-элементами
